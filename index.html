[+] ESTRUTURA E COMANDOS ESSENCIAIS

Domine os cmdlets e aliases fundamentais para navegar e auditar ambientes Windows:

Cmdlet / Alias Descrição Operacional
Get-Location / pwdExibe o diretório atual de trabalho.
Set-Location / cdNavega entre os diretórios do sistema.
Get-ChildItem / lsLista arquivos e pastas (use -Force para ocultos).
Get-Content / catLê e exibe o conteúdo de um arquivo de texto.
Select-String / grepBusca por padrões de texto específicos em arquivos.
Get-CommandLista todos os comandos disponíveis no ambiente atual.
Get-Help / manExibe o manual de ajuda de um comando específico.
Get-Process / psInspeciona os processos ativos em execução no sistema.
Get-ServiceExibe o status dos serviços do Windows.
Copy-Item / cpCopia arquivos ou diretórios para outro local.
Move-Item / mvMove ou renomeia arquivos ou diretórios.
Remove-Item / rmRemove permanentemente arquivos ou pastas.
New-Item / mkdirCria um novo item (arquivo ou diretório) no sistema.
Clear-Host / clsLimpa a tela do console de comandos.
Get-HistoryExibe o histórico de comandos executados na sessão.
